# Priscila's Portfolio

A front-end portfolio built to demonstrate how I use AI responsibly across
planning, implementation, testing, and documentation — not just to ship code,
but to ship code someone else could maintain.

I'm a first-semester Information Systems student and a Front-end AI Engineer
intern at Flyrank. This repo is both my portfolio site and a record of that
process: see [`/work`](./app/work) for the full case study, or the
[live preview](#) for the deployed version.

Next.js 14 (App Router) · TypeScript · Tailwind CSS · Server Components by default
